# Módulo 3: A01:2025 – Broken Access Control (incluye SSRF)

**Objetivos de aprendizaje:**
- Comprender la diferencia entre autenticación y autorización.
- Identificar y explotar Broken Object Level Authorization (BOLA / IDOR).
- Explotar privilege escalation (horizontal y vertical).
- Entender y explotar Server-Side Request Forgery (SSRF).
- Implementar controles de acceso seguros (RBAC, ownership checks, allowlists).
- Relacionar con OWASP API Security Top 10 (API1:2023 y API3:2023).

**Duración recomendada:** 6 horas (1.5 horas teoría + 4.5 horas labs).

**Posición en OWASP Top 10:2025:** #1 (la vulnerabilidad más crítica).

**Referencias:**
- OWASP Top 10 2025 – A01: Broken Access Control
- OWASP API Security Top 10 – API1: Broken Object Level Authorization
- OWASP Cheat Sheet: Authorization
