# Secure Software Development Lifecycle (Secure SDLC)

## Fases recomendadas:
1. **Requirements** → Incluir requisitos de seguridad
2. **Design** → Threat Modeling (STRIDE)
3. **Implementation** → Secure Coding + SAST
4. **Verification** → DAST, IAST, Pentesting
5. **Release** → SBOM + SCA
6. **Maintenance** → Monitoring + Patch Management

**Recomendación:** Adoptar OWASP SAMM o BSIMM como marco de madurez.
