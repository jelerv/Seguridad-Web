# Módulo 6: A04:2025 – Cryptographic Failures

**Objetivos de aprendizaje:**
- Entender las Cryptographic Failures según OWASP Top 10:2025.
- Identificar fallos comunes en el uso de criptografía en aplicaciones web y APIs.
- Diferenciar entre algoritmos criptográficos débiles y fuertes.
- Implementar almacenamiento seguro de contraseñas (hashing + salting).
- Gestionar correctamente claves, tokens y datos sensibles en tránsito y en reposo.
- Configurar correctamente TLS/HTTPS y evitar exposición de datos sensibles.
- Relacionar con OWASP API Security Top 10 (especialmente Broken Authentication y Sensitive Data Exposure).

**Duración recomendada:** 5 horas (1.5 horas teoría + 3.5 horas labs).

**Posición en OWASP Top 10:2025:** #4

**Consecuencias comunes:**
- Robo de credenciales
- Decifrado de datos sensibles
- Session hijacking
- Exposición de información confidencial
