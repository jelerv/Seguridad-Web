# Módulo 9: A07:2025 – Identification and Authentication Failures

**Objetivos de aprendizaje:**
- Entender las fallas más comunes en identificación y autenticación según OWASP Top 10:2025.
- Identificar y explotar ataques como Credential Stuffing, Brute Force, Session Hijacking y JWT weaknesses.
- Implementar autenticación segura con MFA, passwordless y gestión correcta de sesiones/tokens.
- Diferenciar entre autenticación y autorización.
- Aplicar buenas prácticas para APIs (OAuth 2.1, OpenID Connect).
- Relacionar con OWASP API Security Top 10 (API2:2023 – Broken Authentication).

**Duración recomendada:** 5 horas (1.5 horas teoría + 3.5 horas labs).

**Posición en OWASP Top 10:2025:** #7

**Impacto típico:**
- Robo de cuentas
- Session hijacking
- Acceso no autorizado a datos sensibles
- Privilege escalation
