# Módulo 10: A08, A09, A10:2025 + Seguridad Avanzada en APIs

**Objetivos de aprendizaje:**
- Comprender las tres categorías finales del OWASP Top 10:2025.
- A08: Software and Data Integrity Failures (incluye deserialización insegura y CI/CD compromises).
- A09: Security Logging and Monitoring Failures.
- A10: Mishandling of Exceptional Conditions (nuevo en 2025).
- Dominar los riesgos específicos de APIs según OWASP API Security Top 10 2023.
- Identificar y mitigar ataques avanzados en APIs modernas (GraphQL, REST, rate limiting, etc.).
- Integrar las mejores prácticas de seguridad en el desarrollo de APIs.

**Duración recomendada:** 6 horas (2 horas teoría + 4 horas labs prácticos).

**Categorías cubiertas:**
- **A08:2025** – Software and Data Integrity Failures
- **A09:2025** – Security Logging and Monitoring Failures
- **A10:2025** – Mishandling of Exceptional Conditions (nuevo)

**Relación con API Security:**
- API2: Broken Authentication
- API3: Broken Object Property Level Authorization
- API4: Unrestricted Resource Consumption
- API6: Server-Side Request Forgery (ya cubierto en M3)
- API8: Security Misconfiguration
- API9: Improper Inventory Management

- OWASP Top 10 2025
