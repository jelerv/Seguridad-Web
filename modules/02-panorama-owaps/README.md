# Módulo 2: OWASP Top 10:2025 – Panorama General

**Objetivos de aprendizaje:**
- Conocer la historia y evolución del OWASP Top 10 (2017 → 2021 → 2025).
- Entender los cambios principales en la versión 2025.
- Comprender la metodología de clasificación del OWASP Top 10.
- Diferenciar riesgos de aplicaciones web tradicionales vs riesgos en APIs modernas.
- Conocer la relación entre OWASP Top 10 Web y OWASP API Security Top 10 2023.
- Identificar cómo aplicar el Top 10 en proyectos reales de desarrollo.

**Duración recomendada:** 2 horas (1.5 horas teoría + 0.5 horas discusión y laboratorio).

**Contenido principal:**

### Evolución del OWASP Top 10
- 2017 → 10 riesgos más comunes
- 2021 → Mayor énfasis en datos y APIs
- **2025** → Versión actual (publicada en 2025):
  - Nueva categoría: **A03:2025 – Software Supply Chain Failures**
  - Nueva categoría: **A10:2025 – Mishandling of Exceptional Conditions**
  - SSRF se integra en **A01: Broken Access Control**
  - **Security Misconfiguration** sube al puesto #2
  - **Injection** baja al #5

### Relación con APIs
- OWASP API Security Top 10 2023 complementa el Web Top 10.
- Muchas vulnerabilidades se superponen (Broken Object Level Authorization, etc.).
