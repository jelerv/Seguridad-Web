# Módulo 7: A05:2025 – Injection

**Objetivos de aprendizaje:**
- Entender el concepto de vulnerabilidades de inyección según OWASP Top 10:2025.
- Identificar y explotar SQL Injection (clásica, union-based, blind, error-based).
- Conocer otras inyecciones comunes: NoSQL, Command Injection, LDAP.
- Aplicar las mejores prácticas de prevención (parameterized queries, ORM, validación).
- Realizar pruebas automatizadas con sqlmap.

**Relación con APIs:** Inyección en endpoints REST (parámetros de query, body JSON, GraphQL).

**Duración recomendada:** 5 horas (1 hora teoría + 4 horas labs).

**Referencias oficiales:**
- OWASP Top 10 2025 – A05: Injection
- OWASP SQL Injection Prevention Cheat Sheet
