# Módulo 7: A05:2025 – Injection

**Objetivos de aprendizaje:**
- Comprender las vulnerabilidades de inyección según OWASP Top 10:2025.
- Identificar y explotar diferentes tipos de inyección: SQL, NoSQL, Command, LDAP y XML.
- Explotar SQL Injection clásica, Union-based, Blind y Error-based.
- Aplicar técnicas modernas de prevención (prepared statements, ORM, validación estricta).
- Realizar pruebas automatizadas con sqlmap.
- Entender el impacto de la inyección en APIs modernas (REST y GraphQL).

**Duración recomendada:** 5 horas (1 hora teoría + 4 horas labs).

**Posición en OWASP Top 10:2025:** #5

**Referencias oficiales:**
- OWASP Top 10 2025 – A05: Injection
- OWASP SQL Injection Prevention Cheat Sheet
- OWASP Testing Guide v4 – Input Validation Testing
