# Módulo 11: Pruebas de Seguridad y Mitigación de Vulnerabilidades

**Objetivos de aprendizaje:**
- Conocer las metodologías de pruebas de seguridad en aplicaciones web y APIs (OWASP Testing Guide v4).
- Dominar el uso de herramientas profesionales: Burp Suite, OWASP ZAP, sqlmap, Nuclei, jwt_tool.
- Realizar pentesting manual y automatizado de aplicaciones web y APIs.
- Diferenciar entre SAST, DAST, IAST y SCA.
- Implementar estrategias de mitigación efectivas.
- Integrar seguridad en pipelines CI/CD (DevSecOps).
- Generar reportes profesionales de vulnerabilidades.

**Duración recomendada:** 6 horas (1.5 horas teoría + 4.5 horas labs intensivos).

**Metodologías cubiertas:**
- OWASP Testing Guide v4
- OWASP Web Security Testing Guide
- PTES (Penetration Testing Execution Standard)
- Bug Bounty methodologies

**Herramientas principales:**
- Burp Suite Community
- OWASP ZAP
- sqlmap
- Nuclei
- jwt_tool, ffuf, dirsearch
- Trivy, Dependency-Check (SCA)

**Referencias oficiales:**
- OWASP Testing Guide v4
- OWASP Top 10 2025
- OWASP API Security Testing Guide
