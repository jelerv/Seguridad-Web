# Curso: Seguridad en Aplicaciones Web y APIs  
**Basado en OWASP Top 10:2025 + OWASP API Security Top 10 2023**

Repositorio oficial del curso práctico hands-on.

## Estructura
Cada módulo contiene:
- `README.md` → Teoría y objetivos
- `vulnerable/` → Código inseguro (para explotar en clase)
- `secure/` → Código corregido
- `lab-guide.md` → Guía paso a paso del laboratorio
- `exploits/` → Scripts de explotación
- `docker-compose.yml` (opcional por módulo)

## Cómo empezar
1. Lee `setup.md`
2. `docker compose up -d`
3. Navega a `modules/03-broken-access-control/` y comienza el primer lab

¡Bienvenidos al curso!  
Desarrollado para dictar con ejemplos reales en Python/Flask y Node.js.
